# Block Craft — Godot Game

Мобильная головоломка на Godot 4. Игрок укладывает блоки в силуэт формы.

## Стек
- Godot 4 (GDScript)
- Мобильная игра (iOS / Android)

## Структура
- `scripts/` — вся логика (GDScript .gd файлы)
- `scenes/` — сцены (.tscn файлы)
- `assets/` — арты, звуки

## Ключевые скрипты
- `GameState.gd` — глобальное состояние: инвентарь, build_stage, hub_trees_chopped
- `Hub.gd` — главный хаб, дом строится поэтапно, STAGE_LEVELS = {1:[101,102], 2:[103,104], 3:[105,106], 4:[107]}
- `PuzzleLevel.gd` — логика уровней-головоломок
- `GridBoard.gd` — игровая сетка
- `Piece.gd` — блоки/фигуры
- `HouseBuilder.gd` — строительство дома в хабе
- `MainMenu.gd`, `LevelSelect.gd` — навигация
- `ShopScreen.gd`, `CustomisationScreen.gd` — магазин и кастомизация
- `AdManager.gd` — реклама
- `SoundManager.gd` — звук

## Нарратив (Story Mode)
Персонажи:
- **Фермер** — молчаливый NPC, купил заброшенный участок
- **Енот (маскот)** — болтливый, добрый, живёт на участке, гид по игре

Поток:
1. Уровень 1 (форма Топор) → inventory["axe"] → деревья в хабе стали интерактивными
2. Срубил деревья → появляется фундамент + стрелка
3. Строительные уровни (101-107): Фундамент → Каркас → Крыша → Стены
4. После фундамента → появляется Енот с диалогом
5. Дом готов → Енот предлагает: Забор / Дорожка / Огород

## Моменты появления енота
- Первое знакомство (после фундамента)
- Перед каждым уровнем — реплика по теме
- Комбо 3+ — радуется
- Новая постройка — комментирует
- Ежедневный вход — выдаёт награду
- Давно не заходил — скучает

## Статус
- Механика головоломок работает
- Story mode / нарратив — в разработке
- Арты енота и фермера — ждём от дизайнера
- Диалоговая система — не реализована
